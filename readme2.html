Project Overview

What this project does

Football Scouting ML predicts player market values based on performance and contextual data. The goal is not to replicate Transfermarkt exactly, but to build a transparent, data-driven lens on valuation that can support scouting and investment decisions.

Core Idea

The model takes structured player attributes (performance statistics, age, position, league, etc.) and outputs an estimated market value in euros. This allows clubs, analysts, and agents to:

  • Identify undervalued and overvalued players.
  • Compare players across leagues and seasons.
  • Quantify how different attributes impact valuation.

Main Objectives

  • Build an end-to-end ML pipeline from raw data to predictions.
  • Evaluate model performance across leagues and player segments.
  • Provide model interpretability for individual players.

Scope & Assumptions

  • Focus on the Big-5 European leagues (Premier League, La Liga, Bundesliga, Serie A, Ligue 1).
  • Player-season level observations, covering multiple seasons.
  • Market value from Transfermarkt (non-official but widely used baseline).
  • Performance data from match statistics (minutes, goals, assists, etc.).

Data & Processing

From raw data to clean training set

The project integrates multiple data sources, handles missing values and inconsistencies, and produces a clean feature matrix ready for training.

Data Sources

  • Market Values: Transfermarkt player market values per season.
  • Performance Stats: Match-level or season-level statistics (minutes, appearances, goals, assists, cards, etc.).
  • Contextual Data: Player age, position, league, club, possibly contract length and nationality.

Dataset Size (Example)

  • ~N players across 5 major European leagues.
  • ~M player-season observations (multiple seasons).
  • Dozens of engineered features capturing performance, age profile, etc.

Target Variable

The target is the player market value in euros for a given season. To stabilize training, the model can predict a log-transformed market value and then map it back to euros for interpretation.

Processing Pipeline

The following steps are implemented in the data pipeline:

  • Load raw scraped / downloaded data into Pandas DataFrames.
  • Merge multiple tables (players, stats, leagues) on keys like player ID and season.
  • Handle missing values and outliers (e.g., imputation, filtering).
  • Engineer domain-specific features (per-90 stats, age buckets, etc.).
  • Split into train/validation/test with time-aware logic if needed.
  • Scale / encode features where appropriate.

Interpretability

Why does the model value a player that way?

Beyond raw metrics, the project uses SHAP values and feature importance plots to explain model decisions at both the global and individual levels.

Global Feature Importance

SHAP summary plots and feature importance charts highlight which variables contribute most strongly to market value predictions. Typical top features include:

  • Minutes played and games started.
  • Goals and assists (especially for attacking positions).
  • Age (peak age range vs. younger or older players).
  • League and club strength.

Player-Level Explanations

For a given player, the SHAP values can be visualized as a bar chart showing how each feature pushes the predicted value up or down relative to a baseline. This can be turned into a simple "explanation card":

  • Positive contributions (for example, high minutes, strong goal output) raise the predicted value.
  • Negative contributions (for example, low playing time, advanced age) reduce the predicted value.

This is particularly valuable for scouting, as it makes the model's reasoning transparent and allows comparison with subjective judgment.

Insights

What the model teaches us about player valuation

Beyond prediction accuracy, the model reveals how different factors drive market values across leagues, age groups, and playing positions.

Peak age window matters
Players in the typical "prime" age band (for example, 23-28) exhibit higher values for similar performance compared to much younger or older players.
Playing time is a strong signal
Minutes played and share of available match time are among the strongest predictors of market value, reflecting trust from the coaching staff.
League effects and visibility
The same performance statistics often map to different values in different leagues, reflecting competition level, commercial visibility, and club budgets.
Attacking output carries a premium
Goals, assists, and key passes tend to drive valuations more aggressively for attacking players, while defensive metrics matter more for defenders and defensive midfielders.
Roadmap

Planned improvements and extensions

The current version provides a solid baseline for market value prediction. Future work focuses on richer data, better modeling, and stronger real-world integration.

Model and Data Enhancements

  • Add granular injury data and contract length where available.
  • Incorporate advanced metrics (xG, xA, pressing, progressive passes).
  • Refine league and club strength indicators (for example, Elo ratings).
  • Experiment with alternative models (CatBoost, LightGBM, ensembles).

Product and Usability

  • Deploy the model behind an API endpoint for external use.
  • Build a richer web frontend with player search and comparison views.
  • Integrate multi-season trajectories to estimate future value evolution.
  • Support a workflow for scouts to annotate and override model outputs.
